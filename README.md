# Superstore Sales Analysis Dashboard

## Overview
Analysed a 9,800-row retail sales dataset sourced from Kaggle to identify revenue trends, 
top-performing products, and regional sales patterns using Power BI and Excel.

## Tools Used
- Microsoft Power BI
- Microsoft Excel
- Dataset: Kaggle (Sample Superstore)

## Key Insights
- West region recorded the highest sales across all 4 regions
- Technology was the top performing category at 36.59% of total sales
- Sales showed a consistent upward trend from 2015 to 2018
- Consumer segment dominated at 50.76% of total sales
- Canon imageCLASS was the single highest selling product
- Phones led all sub-categories in revenue
